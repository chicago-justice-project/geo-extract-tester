Get the validation data.
The format is simple. Distinct words are newline separated, and documents are separated by two newlines.

You need to upload a file with the same number of lines, where each line is a number between 0 and 1 inclusive representing the probability that word contains location information. Where the original file contains empty lines, your uploaded file should also contain empty newlines.

Below is a short example python script to load the data and make random guesses. 
with open('validation.txt') as f:
  s = f.read()

import random
def guess(word):
  if word:
    return str(random.random())
  else:
    return ''

with open('guesses.txt', 'w') as f:
  for line in s.split('\n'):
    f.write(guess(line) + '\n')
